CryptoChat is a browser extension that adds end-to-end encryption to Discord, Slack, WhatsApp Web, Telegram, Instagram, X/Twitter, and Facebook Messenger. Your keys never leave your device.

Chrome / Brave / Edge Firefox
Discord Slack WhatsApp Web Telegram Web Instagram DMs X / Twitter Facebook Messenger
🔒
AES-256-GCM
Every message encrypted with a unique random key. Platforms only see ciphertext.
🔑
Keys stay local
Your private key never leaves your browser. Stored in IndexedDB, never synced.
👥
Group encryption
One message, multiple recipients. Each gets their own encrypted key slot.
🗝️
GPG compatible
Import OpenPGP public keys from Kleopatra or any GPG keychain.
